# Prosperity360 Foundation Website

A comprehensive Flask-based website for the Prosperity360 Foundation, showcasing their global strategy and development initiatives across Africa and the Global South.

## Features

### Core Features

- **Responsive Design**: Mobile-first approach with full responsiveness
- **Dark/Light Theme Toggle**: User preference-based theme switching
- **Interactive Navigation**: Smooth scrolling and mobile-friendly menu
- **Multi-page Structure**: 6 main pages covering all foundation aspects
- **Form Handling**: Contact, donation, and volunteer forms with validation
- **Dynamic Content**: Interactive elements and animations

### Pages Included

1. **Home** - Landing page with hero section, mission statement, and CTAs
2. **About Us** - Foundation identity, philosophy, and multi-dimensional framework
3. **Programs** - Detailed view of 6 strategic pillars and delivery architecture
4. **Impact** - Impact metrics, roadmap, CSO alliance, and innovation platforms
5. **Get Involved** - Donation, partnership, volunteering, and advocacy opportunities
6. **Contact** - Contact information, regional offices, and FAQ

### Technical Features

- **Flask Backend**: Python-based web framework
- **Pure CSS/JS**: No external CSS/JS frameworks (except Font Awesome)
- **Theme System**: CSS variables for easy theme management
- **Form Validation**: Client and server-side validation
- **Accessibility**: ARIA labels, semantic HTML, keyboard navigation
- **Performance**: Optimized assets and lazy loading

## Installation & Setup

### Prerequisites

- Python 3.7+
- pip (Python package manager)
